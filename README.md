# Quizify

## Description:
 This is quizify, a clean and fully responsive web-based quiz application. you can test your programming knowledge,geography knowledge, etc. This app has you covered with customizable categories and a ticking timer to keep things exciting.

# Why i made this:
 i wanted to build a highly engaging project that moves away from boring forums and brings a dynamic user experience. the goal was to practice cleanly manipulating the DOM in real-time, structuring application state without relying on complex frameworks, and creating smooth UI transitions that keeps users hooked.

# How it works:
 Quzify is a fast, single page application that swaps views between setup, gameplay to results. when you select a category, the app randomly serves questions sourced from an questions.js file i found on internet, this handles a 15 second countdown, and validates your answers instantly. it tracks your live score and blocks double clicking, sending you straught to a custom final scorecard once your rounds are complete.

## Tech Stack:
 * HTML: Structures the layout of the configuration view, question modules, and final scorecard
 * CSS: built with a clean custom layout utilizing Flexbox and grid, featuring a smooth mobile experience.
 * Javascript: handles all core application logic including dynamic question indexing, interval timers,  click state management, and a realtime DOM manipulation.
